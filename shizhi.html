This project is a learning APP designed for the elderly. Because we find that with the development of the smart age, the elderly are using mobile phones more and more. However, many elderly people can't even do the most basic mobile phone operations, such as paying utility bills, adding friends, etc.. So, we designed a learning software. It aims to let the elderly learn some basic mobile phone operations through simple buttons and interfaces.
